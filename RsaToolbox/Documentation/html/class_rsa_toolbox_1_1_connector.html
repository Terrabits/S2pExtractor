Detailed Description

Handles connector type and gender.

Rohde & Schwarz Vector Network Analyzers come with a default list of connector types. In addition, user-defined connector types are possible. The Connector class enables both system- and user-defined connector types to be handled by the same code without special treatment.

Convenience functions such as isMale() and getMatingConnector() are provided for processing connectors.

Several display functions are provided for generating user-friendly text representations such as displayType(), displayGender() and displayText().

Several static functions are also provided for handling a QVector of Connector objects.

bool isSingleConnectorType ( QVector< Connector > &  connectors)
static

Compares the type of all Connector objects in connectors.

Certain VNA operations require all connector types be the same. For example, TOSM calibration is only available when the system uses connectors of the same type. This function, then, can be useful in determining if TOSM calibration is possible.

Example:

QVector<Connector> connectors;
connectors.append(Connector(N_50_OHM_CONNECTOR, MALE_GENDER));
connectors.append(Connector(N_50_OHM_CONNECTOR, FEMALE_GENDER));
bool isSameType = isSingleConnectorType(connectors); // This value is true

Counts the number of female connectors in connectors.
